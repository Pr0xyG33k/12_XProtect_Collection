<!-- REQUIREMENTS -->
<h2>requirements</h2>

Clone the repository :
```powershell
git clone https://github.com/Pr0xyG33k/XProtect_Milestone02.git
```

Navigate to the project directory :
```powershell
cd XProtect_Milestone02/main
```

<!-- USAGE -->
<h2>usage</h2>

Execute the script :
```powershell
.\Milestone_XProtect02.ps1
```
